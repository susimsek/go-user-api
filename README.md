# Go User Api
> This is user rest endpoint example using go language.

<img src="https://github.com/susimsek/go-user-api/blob/master/images/golang-mysql.png" alt="Go Mysql User Api" width="50%" height="50%"/> 

## Prerequisites

* Go 1.14.x
* Docker 19.03.x
* Docker Compose 1.25.x

## Installation

```sh
docker-compose up -d
```

## Used Technologies

* Gorm
* Gin
* Mysql
